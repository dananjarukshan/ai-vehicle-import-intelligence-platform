# AI Vehicle Import Intelligence Platform

## Overview

An AI-powered platform that automates vehicle auction analysis, import price estimation, and market intelligence generation using OCR, workflow automation, and analytics.

## Features

- Auction screenshot upload
- AI OCR extraction
- Vehicle database management
- Import cost estimation
- Selling price prediction
- Market trend analytics
- Dashboard visualization

## Technology Stack

### Frontend
- React.js
- Tailwind CSS

### Backend
- Node.js
- Express.js
- PostgreSQL

### Automation
- n8n

### AI
- OpenAI GPT-4o Vision
